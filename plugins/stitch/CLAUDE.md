# Stitch MCP

Google Stitch is an AI-powered UI design tool accessible via the Model Context Protocol.

## Available Tools

- **Project Management**: create_project, get_project, list_projects
- **Screen Management**: list_screens, get_screen
- **AI Generation**: generate_screen_from_text, edit_screens, generate_variants

## Usage

Use Stitch tools to create and manage UI designs. When generating screens, provide clear descriptive prompts for best results.

Models available for generation: `GEMINI_3_FLASH` and `GEMINI_3_1_PRO`.
